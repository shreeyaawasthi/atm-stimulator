# atm-stimulator
A simple ATM Simulator project developed in Python. This program simulates basic ATM operations such as checking account balance, depositing money, withdrawing money, and exiting the system.

📌 Features
Check Account Balance
Deposit Money
Withdraw Money
Input Validation
User-Friendly Menu Interface
Continuous Transactions Until Exit
🛠️ Technologies Used
Python 3
